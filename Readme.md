# Lexer - implemented with JavaScript

## Installation

### Rquirements
- NodeJS

### 1. Clone Repo
```sh
git clone https://github.com/6f5/lexer-js
```

### 2. Install Dependencies
```sh
npm install
```

### 3. Run Tests
```sh
npm test
```

### 4. Run Repl
```sh
node main.js
```
